HTML Heading Elements
---------------------
- A HTML tag refers to the text enclosed w/in a pair of "< >"
    - Often written as a pair of starting & ending tag
        - Starting tag  = < ... >
        - Ending tag    = </ ... >
        - e.g. <h1>Hello there!</h1>


- A HTML element refers to
    - The starting HTML tag
    - The content
    - The ending HTML tag 

    - NOTE: Content is "sandwiched" between the starting & ending HTML tag
        - e.g. <h1>Hello there!</h1>


- The HTML Heading element
    - A HTML element whose purpose is to convey a "level of importance"
        - Taken inspiration from traditional book binding

    - HTML has six (6) Heading elements
        - <h1>  = highest level of importance
        - <h6>  = lowest level of importance

    - A HTML Heading element is a "block level" element
        - It renders a new line on the browser
        - It takes up the ENTIRE WIDTH available on their containing block

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements)

    - IMPORTANT:
        - AVOID multiple <h1> elements w/in a single web page 
            - Limit to using ONLY one (1) <h1> element per web page

        - DO NOT SKIP a Heading element level 
            - Go through ALL levels from <h1> to <h6>

    - Example:
        - Contents of a single web page w/ Headling elements

        <h1>Papa's Pizza</h1>
            
        <h2>The best pizza in the multiverse</h2>

        <h3>Lorem ipsum</h3>
