What Is HTML
------------
- HTML stands for "HyperText Markup Language"
    - It defines the content & structure of a website 

    - The "HyperText" part refers to
        - The pieces of text that can be linked to other documents w/in a website

    - The "Markup Language" part refers to
        - The HTML tags that is used on "marking up" a document (i.e. web page)
